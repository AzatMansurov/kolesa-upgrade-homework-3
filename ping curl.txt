ping - команда которая запускает утилиту проверки соединении между двумя точками в сети

64 bytes from 142.250.186.206 - размер пакета отправленный с адреса 142.250.186.206
icmp_seq=0 - порядковый номер пакета
ttl=113 - время жизни пакета. В данном случае количество итеарции отправлении
time=123 ms - время которая ушла с момента отправления до получения


curl - команда для отправки запросов

HTTP/2 200 - версия протокола и статус
server: nginx - используется сервер nginx основе
-I - выводит только заголовки
